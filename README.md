# IA Mentor de Carreira

> Sistema multi-agente com inteligência artificial para ajudar pessoas a descobrirem caminhos profissionais em tecnologia, identificando interesses, sugerindo áreas de atuação e indicando habilidades importantes para o desenvolvimento da carreira.

## Visão geral

O IA Mentor de Carreira organiza a jornada de evolução profissional em 3 fases de agentes inteligentes:

| Fase | Agente | Objetivo |
|---|---|---|
| 1 | Diagnóstico | Entender perfil, interesses, experiência e meta de carreira |
| 2 | Planejamento | Gerar roadmap, mapa de skills e plano de estudo prático |
| 3 | Execução | Acompanhar progresso, ajustar rota e preparar para entrevistas |

## Como usar

1. Leia `plano-completo/00-visao-geral.md` para entender o perfil alvo e a estratégia anual.
2. Siga `plano-completo/03-roadmap-90-dias.md` para executar a primeira fase de estudos.
3. Desenvolva o projeto prático em `plano-completo/04-projeto-portfolio.md`.
4. Use `plano-completo/05-roteiro-entrevistas.md` para treinar respostas e simular entrevistas.
5. Consulte `GIT-TUTORIAL.md` para consolidar fluxo Git e colaboração.
