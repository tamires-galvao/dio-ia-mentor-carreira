# 03 — Roadmap de 90 dias (8h/semana)

## Diretriz geral

- Carga semanal: **8 horas**
- Divisão sugerida: **40% teoria (3h12)** e **60% prática (4h48)**

## Mês 1 (Dias 1–30): Base técnica e organização

### Semana 1
- Tópicos: Java 17, records, streams, boas práticas de código limpo.
- Tarefas: revisar fundamentos e criar mini API REST.

### Semana 2
- Tópicos: Spring Boot 3, validação, tratamento global de erros.
- Tarefas: evoluir API com camadas e DTOs.

### Semana 3
- Tópicos: Angular 17, componentes standalone, serviços e rotas.
- Tarefas: criar frontend para consumir API.

### Semana 4
- Tópicos: Testes backend/frontend.
- Tarefas: adicionar testes unitários principais.

## Mês 2 (Dias 31–60): IA aplicada e integração

### Semana 5
- Tópicos: fundamentos de LLM, embeddings, chunking.
- Tarefas: estudo de RAG e desenho de arquitetura.

### Semana 6
- Tópicos: Spring AI e integração com provedor de modelo.
- Tarefas: criar endpoint de perguntas e respostas.

### Semana 7
- Tópicos: PostgreSQL + pgvector.
- Tarefas: persistir documentos e implementar busca vetorial.

### Semana 8
- Tópicos: integração frontend com assistente IA.
- Tarefas: fluxo conversacional com histórico simples.

## Mês 3 (Dias 61–90): Produção e posicionamento

### Semana 9
- Tópicos: Docker e ambiente local reproduzível.
- Tarefas: containerizar backend, frontend e banco.

### Semana 10
- Tópicos: AWS e deploy inicial.
- Tarefas: publicar versão funcional do projeto.

### Semana 11
- Tópicos: observabilidade e qualidade.
- Tarefas: logs estruturados e testes de regressão.

### Semana 12
- Tópicos: preparação de portfólio e entrevistas.
- Tarefas: gravar demo, atualizar README e treinar respostas.

## Checklist de progresso

- [ ] Mês 1 concluído com base sólida em Java + Angular
- [ ] Mês 2 concluído com RAG funcionando ponta a ponta
- [ ] Mês 3 concluído com deploy e narrativa de carreira pronta
- [ ] Projeto de portfólio publicado e documentado
