# 06 — Trilha DIO recomendada

## Bootcamp sugerido

**Coding The Future: Java AI Developer**

## Por que combina com este perfil

- Une base sólida em Java com aplicação prática de IA.
- Acelera transição para projetos modernos com foco em empregabilidade.
- Permite evolução consistente para responsabilidades de nível sênior.

## 5 próximos passos para se inscrever

1. Criar ou atualizar conta na plataforma da DIO.
2. Buscar o bootcamp **Coding The Future: Java AI Developer**.
3. Ler ementa, carga horária e pré-requisitos.
4. Planejar agenda semanal (8h) antes de iniciar.
5. Iniciar trilha e registrar progresso no projeto pessoal.

## Trilhas de apoio recomendadas

| Trilha | Objetivo |
